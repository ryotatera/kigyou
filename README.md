# 起業の科学ポータル｜動画プレビュー戦略 プロトタイプ

`SSP-MEDIA-VIDEO-RD-v1.0` 要件定義の中核（§3 型 B チャプターカット、§5 プレイヤー UI/UX、
B-01〜B-05 5 原則ペイウォール）の動作プロトタイプ。

## 起動

```bash
npm install
npm run dev
# → http://localhost:3050
```

## 操作

1. ヘッダーの「ログイン（プロト用）」で会員状態を切替
2. 動画詳細ページで再生開始
3. 公開区間終端の **30 秒前** から右上にカウントダウン ピル
4. 公開区間終端で動画停止 + ペイウォール オーバーレイ
5. メール入力 → 登録（300ms で自動ログイン）→ その場で再生再開
